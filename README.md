# Acest site este util pentru a copia cartelele optice de interfon folosind o imprimanta 3D.
## Doar introdu codul cartelei (testat cu cartela pe 12 biti, probabil merge si cu 14 biti) si se va genera un fisier STL cu o copie a cartelei pentru printare 3D.

### Cartelele optice de obicei se copiaza folosind o folie de plastic si o imprimanta laser pe care se printeaza codul de bare. Aceasta metoda (printarea 3D) este mai prietenoasa cu mediul, deoarece nu genereaza exces de material (restul foliei de plastic la metoda normala), foloseste doar cat filament are nevoie.


Pentru a contacta autorul, deschideti un issue si scrieti mesajul acolo. 

*A se folosi doar in scop personal*
